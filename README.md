# Iris-Classification-
Exploratory data analysis and machine learning project for Iris flower classification using KNN and Decision Tree algorithms with visualization and model evaluation.
This project demonstrates the implementation of machine learning classification algorithms to predict the species of an Iris flower based on its physical features.
The project includes exploratory data analysis (EDA), visualization, model training, evaluation, and prediction using the famous Iris dataset.

📌 Problem Statement

Given the following flower measurements:

Sepal Length

Sepal Width

Petal Length

Petal Width

Predict the Iris flower species:

Iris-setosa

Iris-versicolor

Iris-virginica

📊 Dataset Information

Dataset: Iris Dataset (Scikit-learn built-in)

Total Samples: 150

Features: 4 numerical features

Target Classes: 3

🧠 Machine Learning Algorithms Used

K-Nearest Neighbors (KNN)

Decision Tree Classifier

Both models are trained and evaluated to understand their performance on the dataset.

🔍 Project Workflow

Import required Python libraries

Load the Iris dataset

Perform Exploratory Data Analysis (EDA)

Visualize feature relationships

Split data into training and testing sets

Train KNN and Decision Tree models

Evaluate models using accuracy score

Make predictions on new user input

📈 Visualizations Included

Feature distribution plots

Scatter plots for class separation

Decision Tree visualization (if enabled)

These visualizations help understand how the model learns patterns from data.
